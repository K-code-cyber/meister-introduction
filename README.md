# 🪑 Matrix-Seat-Scheduler

공통수학1의 **행렬과 부등식** 개념을 활용하여 설계한 실생활 문제 해결형 '교실 최적 좌석 배치 알고리즘' 프로젝트입니다.

## 🎯 개발 동기 및 목적
학교 교실이나 자율학습실에서 좌석을 배치할 때, 학생들의 시력 상태, 원하는 짝 배치 등 다양한 제약 조건을 수작업으로 계산하는 비효율성을 발견했습니다. 이를 수학적 행렬로 공간을 정의하고 부등식 제약 조건을 필터링하는 파이썬 알고리즘을 통해 자동화하여 컴퓨팅 사고력을 기르고자 개발하였습니다.

## ✨ 주요 기능
1. **교실 행렬 커스텀**: 행(Rows)과 열(Cols) 설정을 변수화하여 인원수와 교실 크기에 맞춰 유연하게 작동
2. **시력 보호 제약 조건**: 시력이 나쁜 특정 성분의 행 위치 부등식 조건($i \le 2$) 적용
3. **매칭 조건 시스템**: 원하는 짝이 지정된 경우 가로 격자($j-1$ 또는 $j+1$)에 배치되도록 제어
4. **Excel 데이터 연동**: `pandas` 라이브러리를 활용해 사용자 편의성을 높인 데이터 관리 기능

## 🛠️ 기술 스택 및 공학 도구
- **Language**: Python 3
- **Libraries**: NumPy (2차원 행렬 연산), Pandas (데이터 분석 및 엑셀 입출력)
